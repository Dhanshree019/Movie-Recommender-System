# Movie-Recommender-System

Dataset Source :
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

---

## Steps :

1. Install packages
```
pip install -r requirements.txt
```

2. Set up .env file 
```
API_KEY="" // Go to themoviedb and create api key
```

3. movie_recommender_system.ipynb :
- Data Loading
- Data Cleaning
- Data Preprocessing
- Vectorization


4. app.py
- Frontend - Streamlit
- Download .pkl files for movies and similarity and paste it in root level
- Run streamlit application
```
streamlit run app.py
```
